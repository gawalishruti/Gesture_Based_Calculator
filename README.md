# Gesture_Based_Calculator

## Developed by
**Shruti Gawali**  
A passionate AIML engineering student with a strong interest in intelligent systems, real-world problem solving, and modern technologies.

## Objective
To create a calculator that performs arithmetic operations using hand gestures detected through a webcam.

## Technologies Used
- Python  
- OpenCV  
- MediaPipe  
- NumPy  
- Pillow  

## Future Scope
This system can be extended to gesture-based home automation or assistive technologies for physically challenged users.
